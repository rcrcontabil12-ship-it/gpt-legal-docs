# Política de Privacidade

Este GPT foi desenvolvido para auxiliar na geração de documentos societários, como atas, anexos e requerimentos, com base em informações fornecidas voluntariamente pelos usuários.

## Coleta de Dados
Este GPT não coleta, armazena ou compartilha dados pessoais fora do ambiente da plataforma OpenAI.

## Uso das Informações
As informações fornecidas são utilizadas exclusivamente para atender às solicitações do usuário dentro do próprio GPT.

## Compartilhamento de Dados
Nenhuma informação é compartilhada com terceiros.

## Segurança
O tratamento das informações segue as diretrizes de segurança da plataforma OpenAI.

## Contato
Para dúvidas relacionadas a esta política, utilize o próprio ambiente da plataforma.

Última atualização: 2025
